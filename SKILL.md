# 美团热敏打印机 (meituan-printer)

通过 TCP Socket + ESC/POS 协议，向美团热敏打印机发送带格式的任务单/检查单。

## 文件结构

```
meituan-printer/
├── SKILL.md                          # 本文件（AI 指引）
├── scripts/
│   ├── print_to_printer.py           # 核心打印脚本
│   └── init_printer.py               # 引导式配置工具（交互）
├── references/
│   ├── printer-config.md             # 打印机配置参考（IP/类型映射）
│   └── onboarding.md                 # 新手指南（面向用户）
└── config.json                       # 用户配置（运行时生成）
```

## 核心流程：两步走

### 第一步：首次配置（一次性）

用户首次使用（或需要添加/管理打印机）时：

```
python3 scripts/init_printer.py
```

AI 应主动说明引导流程，**不需要**用户自己输入命令。

> **AI 引导要点**：
> - 告知用户将进入交互式配置
> - 解释每个步骤的含义（IP → 端口 → 别名 → 类型 → 备注）
> - 提醒选择"验证连通性"可确保打印机可用

**引导式配置流程**（init_printer.py）：
1. 欢迎 + 环境检查
2. 加载/新建配置文件
3. 添加打印机 → 输入 IP → 智能端口判断（默认 9100）→ 别名（中文）/ 类型 / 备注
4. 保存配置

> **端口智能判断**：用户只提供 IP 时，引导工具会说明"美团热敏打印机默认端口 9100"，直接回车即可使用默认值。

### 第二步：发起打印（日常使用）

```
python3 scripts/print_to_printer.py --name <别名> --title "<标题>" --content "<内容>"
```

## AI 工作流（当用户请求打印时）

### 步骤 1：确认打印目标

按以下优先级确定目标打印机：

| 用户说法 | 对应参数 |
|---------|---------|
| "后厨"、"素菜"、"前厅" | `--name 后厨` 等 |
| "出菜档"、"分菜档"、"前台" | `--name <对应别名>` |
| "IP 是 xxx" / "地址是 xxx" | `--ip <IP>` |
| 未指定 | 查 config.json；无配置则提示先运行 `--setup` |

### 步骤 2：确认任务类型

**必须询问用户**：

> "这个打印任务是**定时任务**还是**一次性打印**？"
> - **一次性**：立即发送到打印机
> - **定时任务**：记录下来，协助用户创建自动化（cron/任务计划程序）

### 步骤 3：确认内容

- **标题（title）**：任务单名称，如"早班A任务单"、"素菜岗检查单"
- **内容（content）**：具体工作事项，多行用 `\n` 分隔
  - 时间节点用 `-------HH:MM-------` 格式
  - 示例：`"-------09:30-------\\n打卡拍照\\n解冻冻货"`

### 步骤 4：执行打印

**立即打印**：调用 print_to_printer.py，捕获结果并告知用户

**定时打印**：
- 记录：打印机别名 + 标题 + 内容 + 触发时间
- 告诉用户定时任务的创建方式（crontab / 任务计划程序）
- 必要时协助生成 crontab 条目

### 步骤 5：输出确认

打印完成后，向用户反馈：
```
✅ 已发送至【后厨】(192.168.3.172:9100)
📄 标题：早班A任务单
⏰ 时间：2026-04-23 14:30
```

## 配置管理命令

| 命令 | 说明 |
|------|------|
| `--setup` | 进入引导配置（添加/删除/查看打印机） |
| `--list` | 列出所有已配置打印机 |
| `--guide` | 显示新手指南 |

## 打印格式说明

打印输出包含：
1. **大标题**（居中 + 双倍大小）：任务单名称
2. **时间戳**（居中 + 倍高）：打印时间
3. **分隔线**：32 个 `=`
4. **正文**（倍高）：具体工作内容，时间节点加粗+倍宽
5. **结尾提示**："请按时完成各项工作"
6. **自动切纸**

## 注意事项

- **Python 环境**：使用 `python3`（macOS/Linux 默认）
- **配置文件**：`scripts/config.json`（自动创建，勿手动编辑）
- **默认端口**：`9100`（美团热敏打印机标准端口，99% 适用）
- **中文编码**：GB18030，确保打印机支持中文热敏纸
- **网络要求**：打印机与电脑在同一局域网内
- **超时设置**：连接超时 5 秒
