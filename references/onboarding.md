# 新手指南 - 美团热敏打印机技能

## ⚠️ 重要：端口说明

> **美团热敏打印机的标准端口是 `9100`**，绝大多数机型默认使用此端口。
>
> - 配置时，如果只提供了 IP 地址，技能会提示使用默认端口 9100，直接回车即可
> - 如果不确定端口是多少，**直接填 9100 准没错**
> - 非美团/非标准机型才需要自定义端口

常见端口参考：

| 端口 | 适用机型 |
|------|---------|
| **9100** | 美团、易联云等大多数热敏打印机 ✅ |
| 80 / 443 | Web 管理后台 |
| 21 / 22 | FTP / SSH（部分商用机型） |

---

## 功能一览

| 功能 | 说明 |
|------|------|
| 打印任务单 | 向指定打印机发送格式化任务单，支持标题+多行内容 |
| 定时打印 | 配合定时任务（cron），在指定时间自动打印 |
| 多打印机管理 | 一台电脑可配置多台打印机，按名称/类型区分 |
| 连通性检测 | 配置时自动验证打印机是否可达 |

## 快速上手（三步）

### 第1步：环境检查

确保已安装 **Python 3**（无需额外依赖库）：

```bash
python3 --version
# 应显示 Python 3.x.x
```

### 第2步：配置打印机（只需一次）

```bash
python3 scripts/init_printer.py
```

引导流程：
1. 选择设备类型（出菜档/分菜档/前台收银等）
2. 输入 IP 地址
3. 输入端口（默认 9100）
4. 设置中文别名（如：后厨、素菜、前厅）
5. 选择是否验证连通性
6. 添加备注（选填）

> 💡 **提示**：同一台电脑可配置多台打印机，分别设置不同别名。

### 第3步：发起打印

**按别名打印（推荐）**
```bash
python3 scripts/print_to_printer.py \
  --name 后厨 \
  --title "早班A任务单" \
  --content "打卡拍照\n解冻冻货\n切盒装鸭血"
```

**直接指定 IP 打印**
```bash
python3 scripts/print_to_printer.py \
  --ip 192.168.3.172 \
  --port 9100 \
  --title "测试单" \
  --content "测试内容"
```

## 定时打印（进阶）

将打印脚本配合系统定时任务（macOS crontab / Windows 任务计划程序）使用：

**macOS 示例：每天 07:00 自动打印早班任务单**
```bash
# 编辑 crontab
crontab -e

# 添加任务（每天早上 7:00）
0 7 * * * /usr/bin/python3 /path/to/print_to_printer.py --name 后厨 --title "早班任务" --content "打卡拍照"
```

> 💡 **提示**：建议使用绝对路径，脚本路径填完整的绝对路径。

## 故障排查

| 现象 | 可能原因 | 解决方案 |
|------|---------|---------|
| 连接超时 | 打印机离线/IP错误 | 检查打印机电源和网络 |
| 连接被拒绝 | 端口错误/打印机未启动 | 确认端口为 9100 |
| 打印乱码 | 编码问题 | 确认打印机支持 GB18030 中文 |
| 找不到别名 | 未配置或拼写错误 | 运行 `--list` 查看已配置的别名 |
