# 新手指南 - 美团热敏打印机技能

## ⚠️ 重要：端口说明

> **美团热敏打印机的标准端口是 `9100`**，绝大多数机型默认使用此端口。
>
> - 配置时，如果只提供了 IP 地址，默认使用端口 9100
> - 如果不确定端口是多少，**直接用 9100 准没错**
> - 非美团/非标准机型才需要自定义端口

常见端口参考：

| 端口 | 适用机型 |
|------|---------|
| **9100** | 美团、易联云等大多数热敏打印机 ✅ |
| 80 / 443 | Web 管理后台 |
| 21 / 22 | FTP / SSH（部分商用机型） |

---

## 功能一览

| 功能 | 说明 |
|------|------|
| 打印任务单 | 向指定打印机发送格式化任务单，支持标题+多行内容 |
| 定时打印 | 支持每天/每周/每月定时自动打印 |
| 多打印机管理 | 一台电脑可配置多台打印机，按名称/类型区分 |
| Web 管理界面 | 浏览器可视化管理，无需终端操作 |
| 连通性检测 | 配置时自动验证打印机是否可达 |

## 快速上手

### 第1步：选择配置方式

有两种方式管理打印机和定时任务，**推荐首选 Web 管理界面**：

| 方式 | 适合场景 | 操作方式 |
|------|---------|---------|
| 🌐 **Web 管理界面**（推荐） | 有浏览器，想直观操作 | 启动后在浏览器中管理 |
| 💬 **对话式配置** | 不方便开浏览器，或偏好打字 | 直接和 AI 对话完成 |

---

### 方式一：Web 管理界面（推荐） 👍

通过浏览器可视化管理打印机、定时任务和打印模板，操作更直观。

#### 启动方式（任选一种）

1. **双击启动脚本**（最简单）：
   - macOS：双击 `scripts/start.command`
   - Windows：双击 `scripts/start.bat`

2. **让 AI 帮你启动**（对话启动）：
   - 直接说："启动打印机管理界面" / "打开 web 后台" / "启动打印机 web 端"
   - AI 会自动启动服务并打开浏览器

3. **终端命令**：
   - macOS：`cd scripts && python3 web_admin.py`
   - Windows：`cd scripts && python web_admin.py`

#### 访问地址

启动后打开浏览器访问：**`http://localhost:5000`**

> 💡 如果 5000 端口被占用（macOS AirPlay 默认占用），会自动切换到 `http://localhost:5001`。

#### 能做什么

| 模块 | 功能 |
|------|------|
| 🖨️ 打印机管理 | 添加/编辑/删除打印机、测试连通性、自定义打印机类型 |
| ⏰ 定时任务 | 创建定时打印任务（每天/每周/每月）、启停控制、立即执行、导出到系统 |
| 📋 模板管理 | 使用预设模板、创建自定义模板 |

> 💡 **提示**：首次使用建议先用 Web 界面配好打印机，后续日常打印直接和 AI 对话即可。

---

### 方式二：对话式配置

不方便打开浏览器时，直接和 AI 对话也能完成所有配置。

#### 配置打印机（只需一次）

告诉 AI 你要配置打印机，AI 会引导你完成：

- 提供打印机的 **IP 地址**（如 `192.168.3.172`）
- 设置一个 **中文别名**（如"后厨"、"素菜"、"前厅酒水"）
- 选择 **打印机类型**（荤菜档/素菜档/前台收银等，类型可自定义）
- AI 会自动检测打印机是否在线，确认后写入配置

> 💡 **提示**：同一台电脑可配置多台打印机，分别设置不同别名。

#### 发起打印

配置完成后，直接告诉 AI 你要打印什么：

- "打印一张早班A任务单到后厨"
- "给前厅酒水打一张测试单"
- "帮我打一张素菜岗检查单，内容是……"

AI 会确认打印目标、标题和内容后自动发送。

## 定时打印（进阶）

定时打印推荐通过 **Web 管理界面** 创建，支持设置频率（每天/每周/每月）、启用/暂停、立即执行测试。

也可以配合系统定时任务使用：

**macOS 示例：每天 07:00 自动打印早班任务单**
```bash
# 编辑 crontab
crontab -e

# 添加任务（每天早上 7:00）
0 7 * * * /usr/bin/python3 /path/to/print_to_printer.py --name 后厨 --title "早班任务" --content "打卡拍照"
```

> 💡 **提示**：建议使用绝对路径，脚本路径填完整的绝对路径。

## 故障排查

| 现象 | 可能原因 | 解决方案 |
|------|---------|---------|
| 连接超时 | 打印机离线/IP错误 | 检查打印机电源和网络 |
| 连接被拒绝 | 端口错误/打印机未启动 | 确认端口为 9100 |
| 打印乱码 | 编码问题 | 确认打印机支持 GB18030 中文 |
| 找不到别名 | 未配置或拼写错误 | 让 AI 列出已配置的打印机，或打开 Web 界面查看 |
| Web 打不开 | 服务未启动 | 双击启动脚本，或让 AI 帮你启动 |
| 5000 端口不可用 | macOS AirPlay 占用 | 自动使用 5001 端口，直接访问 localhost:5001 |
