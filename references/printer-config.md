## 协议说明
- 协议：TCP Socket + ESC/POS 指令集
- 编码：GB18030（中文支持）
- 超时建议：5 秒

## 配置架构
- 打印机配置统一存储在 `config.json`（由 `init_printer.py` 管理）
- `config.example.json` 为模板文件，包含出厂默认配置
- 所有打印机端口统一为 9100（默认值）
- 打印机需与发送端在同一局域网内

## 新增打印机
运行引导配置工具：
```bash
python3 scripts/init_printer.py
```
或直接编辑 `config.json`，按以下格式添加：
```json
{
  "alias": "新名称",
  "ip": "192.168.x.x",
  "port": 9100,
  "type": "设备类型",
  "remark": "备注"
}
```
