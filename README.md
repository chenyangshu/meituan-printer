# 🖨️ 美团热敏打印机技能

通过语音/文字指令，远程控制美团热敏打印机打印任务单、检查单。支持多打印机管理、定时打印、中文别名等功能。

> 适用于餐饮门店的后厨出菜、分菜档、前台收银等多个场景。

---

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 📋 打印任务单 | 向指定打印机发送带格式的任务单/检查单 |
| ⏰ 定时打印 | 设置定时任务，自动在指定时间打印 |
| 🖥️ 多打印机管理 | 一台电脑管多台打印机，按名称/类型区分 |
| 🔧 引导式配置 | 交互式配置工具，无需手动编辑文件 |
| 🌐 中文友好 | 支持中文别名、中文内容、GB18030 编码 |

---

## 🚀 快速上手

### 1. 安装技能

将本仓库克隆到本地：
```bash
git clone https://github.com/YOUR_USERNAME/meituan-printer.git
```

将 `meituan-printer/` 目录放入 WorkBuddy 技能目录即可使用。

### 2. 配置打印机

```bash
cd meituan-printer/scripts
python3 init_printer.py
```

按引导完成：
1. 选择设备类型（出菜档/分菜档/前台收银等）
2. 输入打印机 IP
3. 设置中文别名（如：后厨、前厅、素菜档）
4. 验证连通性 → 保存

> 💡 美团热敏打印机默认端口为 **9100**，直接回车即可。

### 3. 发起打印

```bash
python3 print_to_printer.py --name 后厨 --title "早班任务" --content "打卡\n解冻"
```

## 🔧 常用命令

| 命令 | 说明 |
|------|------|
| `--setup` | 引导配置（添加/管理打印机） |
| `--list` | 列出所有已配置打印机 |
| `--guide` | 显示新手指南 |
| `--name <别名>` | 按别名打印 |
| `--ip <IP> --port <端口>` | 按 IP 直接打印 |
